MOQT Web Client

Browser-based MOQT publisher and subscriber over WebTransport.

  • Publisher — capture camera and publish via MOQT
  • Subscriber — receive and play MOQT video

Setup

  1. Start the relay: cargo run --bin moqt-relay
  2. Start the web server: cd web && npx vite
  3. Open Chrome with: chrome --origin-to-force-quic-on=localhost:4433 --ignore-certificate-errors-spdy-list=localhost
  4. Open the Publisher page, click Start
  5. Open the Subscriber page in another tab, click Start
